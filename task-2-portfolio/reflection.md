# Task 2 – Student Digital Portfolio

## Platforms Set Up

### 1. GitHub
GitHub is a platform for hosting and sharing code using version control (Git).
It is widely used by developers, students, and companies to collaborate on projects.
I created a profile README that introduces who I am, what I am currently learning,
and what I hope to build over the next four years.

### 2. LinkedIn
LinkedIn is a professional networking platform used by employers, recruiters,
and working professionals worldwide. It is the first place internship coordinators
and hiring managers look when they want to learn about a candidate. I filled in
my Education section with my degree, branch, institution, and graduation year.

### 3. Kaggle
Kaggle is a data science and machine learning platform where students and
professionals can practice coding, participate in competitions, and learn from
real datasets. As a CSE (AI) student, this platform will be directly relevant
to my coursework and future career.

## How I Plan to Use These Platforms Over the Next Four Years

Over the next four years, I plan to use GitHub to host every project I build —
starting small with basic Python scripts and gradually working toward larger AI
and machine learning projects. LinkedIn will be my professional front door — I
will keep it updated with internships, certifications, and skills as I grow.
Kaggle will be my practice ground for data science; I plan to attempt at least
one beginner competition per semester to build practical skills alongside
classroom learning. Together, these three platforms will form the foundation
of my digital identity as an engineering student and future professional.
